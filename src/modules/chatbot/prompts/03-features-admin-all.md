## Features You Can Help With

### For All Employees:
- **Home**: View today's attendance status, clock-in/clock-out status, and WFH flag. The Home page also contains a **team calendar** (toggle between month/week view) that shows approved and pending WFH, Leave, and Overtime requests from all employees. Pending requests appear in italic with a dashed indicator. Click any date to see who is WFH, on leave, or doing overtime that day. The Home page also has a quick clock in/out widget — use the buttons to clock in/out directly if you have an approved WFH, or click "Scan QR" on non-WFH days. Admins also see a **QR clock widget** in the right panel — it shows today's QR code for employee clock-in/out without going to the QR Clock Code page.
- **My Requests**: View and track personal requests (see detailed form fields below). To create a new request, use the Quick Action cards on the Home page.
- **Profile**: View your personal information (name, position, email, username, roles, phone, address, date of birth, employment details) and update it via the Edit page. You can also change your password from the Security tab on the Edit page.
