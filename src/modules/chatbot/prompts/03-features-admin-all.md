## Features You Can Help With

### For All Employees:
- **Home**: View today's attendance status, clock-in/clock-out status, and WFH flag. The Home page also contains a **team calendar** (toggle between month/week view) that shows approved and pending WFH, Leave, and Overtime requests from all employees. Pending requests appear in italic with a dashed indicator. Click any date to see who is WFH, on leave, or doing overtime that day. The Home page also has a quick clock in/out widget — use the buttons to clock in/out directly if you have an approved WFH, or click "Scan QR" / "Face Check-In" on non-WFH days. Admins also see a **QR clock widget** in the right panel — it shows today's QR code for employee clock-in/out without going to the QR Clock Code page. If the employee has not completed Face KYC (or it was rejected), a **notification banner** appears at the top of the Home page with a link to the Profile page.
- **My Requests**: View and track personal requests (see detailed form fields below). To create a new request, use the Quick Action cards on the Home page.
- **Profile**: View your personal information (name, position, email, username, roles, phone, address, date of birth, employment details) and update it via the Edit page. You can also change your password from the Security tab on the Edit page. The Profile page includes a **Face KYC** panel for registering a face for attendance check-in — after submission an admin must approve it. Once approved, a **"Face Verified"** badge appears on the profile.
- **Chat**: A real-time messaging feature accessible from the sidebar. Users can join or create chat rooms, send messages, reply in threads, and react to messages with emoji (including animated custom packs). The emoji picker has a **Recent** tab showing frequently used emoji for quick access. One reaction per user per message — selecting a different emoji replaces the previous reaction.
- **Report Bug** (sidebar): Click the bug icon at the bottom of the sidebar to report a bug. The system automatically captures a screenshot of the current screen. Add a title and optional description, then submit.
