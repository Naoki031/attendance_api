## How to Submit a Request

To create a new request, go to the **Home** page.
On the Home page you will see **Quick Action cards** — click the card for the type of request you want:
- 🏠 **WFH** ("Work from home") — WFH request
- 🏖️ **Leave** ("Leave request") — OFF/leave request
- 💻 **Equipment** ("Borrow equipment") — Equipment request
- ⏰ **Clock Forget** ("Missed clock-in/out") — Clock Forget request
- 🔥 **Overtime** ("Overtime request") — Overtime request

Clicking a card opens a dialog form. Fill in the fields and click **"Submit Request"**.

**My Requests page** (in the sidebar) is for viewing your submitted requests only. It has **Edit** (for pending requests) and **Reuse** (copy a past request to submit again) — it does NOT have a create button.
